# Portfolio Tracker

A minimalist, professional investment tracking mobile app built with Flutter.

## Project Structure

- `lib/models/`: Core domain models (`TransactionRecord`, portfolio metrics).
- `lib/screens/`: App screens (`DashboardScreen` with Macro Overview and Portfolio Assets).
- `lib/widgets/`: Reusable minimalist UI components (`MacroOverviewCard`, `PortfolioAssetTile`).
- `lib/services/`: Data and backend integrations (`Firebase`, `Cloud Firestore`).

## Tech Stack
- **Flutter & Dart**
- **Firebase Core & Cloud Firestore**
- **Provider** state management
- **Google Fonts** for minimalist typography
